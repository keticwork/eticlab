# EticLab 🧪
Laboratoire R&D personnel — comprendre chaque brique technologique d'un SaaS, du hardware au business.

## Philosophie
- "Under the hood" d'abord : on ne code rien sans comprendre le concept derrière
- Sécurité par design : chaque module inclut un mode hack
- Documentation visuelle : schémas, captures, démos
